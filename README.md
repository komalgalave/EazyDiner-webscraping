📌 Project Description: EazyDiner Data Extraction, Cleaning & EDA

This project focuses on collecting, cleaning, and analyzing restaurant data from EazyDiner. The goal is to transform raw, unstructured data into a clean and meaningful dataset and then explore it to uncover useful insights.

🔍 Project Workflow

Data Extraction

Extracted restaurant information such as name, location, ratings, cuisines, cost, and other details from the EazyDiner website.

Saved the extracted data in a structured CSV format.

Data Cleaning

Removed duplicate and missing values.

Corrected inconsistent formatting.

Converted data types (like price, ratings, etc.) into usable formats.

Created a clean, ready-to-analyze dataset.

Exploratory Data Analysis (EDA)

Analyzed trends like popular cuisines, cost patterns, rating distributions, and top restaurant categories.

Visualized insights using charts and graphs to better understand the data.

🎯 Objective

The main aim of this project is to build a complete data pipeline—from extraction to analysis—that can help understand restaurant trends and customer preferences.
